# Docker slika za Codespaces za Primijenjenu informatiku

Docker slika je bazirana na CS50 CLI Docker sliki, modificirana za potrebe Primijenjene informatike.

## Komponente

- Bazirana na Ubuntu 24.04.
- Sadrži Java 22.x
- Sadrži Node.js 22.x
- Sadrži Python 3.12.x
- Sadrži Docker 26.x
- Sadrži Rust
- Sadrži Go 1.23.x
